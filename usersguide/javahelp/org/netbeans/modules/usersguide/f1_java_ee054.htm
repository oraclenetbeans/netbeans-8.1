Visual Editor: Edit Group

Use to edit the name of a group assigned to a security role.

This dialog box enables you to edit the following properties:

Element Description

Group Name

 Modifies the name of the group assigned to a security role.

The name of the group must match a name specified on the GlassFish application server.
