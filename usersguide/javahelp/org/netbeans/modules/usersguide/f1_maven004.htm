New Maven Project Wizard: Module Options

Use to set project properties for a new Maven project.

Element Description

NetBeans Version

 Selects the version of the IDE that you want to use as the platform.

Allow OSGi bundles as Dependencies

 Enables you to use OSGi bundles as dependencies of the NetBeans Platform application or modules.

This option is disabled if you select a version of the NetBeans platform that does not support OSGi.

Create Module Project

 Creates a NetBeans Platform module and configure the module as a dependency of the application.

You can specify the name of the module in the text field or use the default name. This option is only available when creating a Maven NetBeans Platform application.
