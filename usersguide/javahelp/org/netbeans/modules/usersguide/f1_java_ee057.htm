Visual Editor: Message Security Bindings

Use to specify the message security entries for the message-security-binding property when using Message Security Binding for a web service endpoint or port.

You also use this dialog box when configuring a message security policy for a port for web service clients.

This dialog box enables you to edit the following properties:

  • Java Method Name (required). Enter this value.
  • Request Authorization Source. Choose from a drop-down menu.
  • Request Authorization Recipient. Choose from a drop-down menu.
  • Response Authorization Source. Choose from a drop-down menu.
  • Response Authorization Recipient. Choose from a drop-down menu.
