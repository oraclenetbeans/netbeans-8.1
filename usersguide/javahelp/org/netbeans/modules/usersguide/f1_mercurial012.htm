Mercurial: Go To Patch Dialog Box

The Go To Patch dialog box enables you to switch to a particular patch in a patch queue series.

To define the patch to be applied to the selected project, file or folder, you need to select the name of a patch in the following field:

Element Description

Patches

 A list of all patches available in a stack.

Names of applied patches display in bold.

Choose Pop all applied patches to remove the applied patches from the top of the stack and update the working directory to undo the effects of the applied patches.
