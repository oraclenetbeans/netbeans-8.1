Inspect and Transform Dialog Box

Use to specify options for the Inspect and Transform refactoring command. Use this command to run the specified inspection on the selected file, package, or project.

Element Description

Inspect

 Inspects the selected file, package, or projects.

Alternatively, click the button to the right to open the Custom Scope dialog box and specify the custom code to be inspected.

Use

 Provides the following options: Configuration and Single Inspection.

    Configuration

 Lists predefined configurations of hints that you can use in source code analysis.

Alternatively, click Manage to display the Manage Inspections dialog box and group a set of hints into a configuration.

    Single Inspection

 Lists single hints that you can use in source code analysis.

Click Browse to see a description of a selected hint.
