New JPA Controller Classes from Entity Class Wizard: Generate JPA Controller Classes

Use to generate JPA controllers based on entity classes in your application.

In this page of the wizard you set the following:

Element Description

Project

 (Read-only) Specify the project in which the JPA controller classes will be located.

Location

 Select the source folder where you want to save the controller classes for the entity classes.

Package

 Select an existing package from the Package drop-down menu or type the name of a new package.
