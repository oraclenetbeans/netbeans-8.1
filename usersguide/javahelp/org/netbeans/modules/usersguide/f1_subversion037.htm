Subversion: Select Tree To Diff Dialog Box

The Select Tree To Diff dialog box enables you to set up a repository folder against which the diff will be performed.

You can specify the following:

Element Description

Repository File

 Specifies the path to the file to be examined. Click Browse to select a branch or tag you need to diff the file to.

Repository Revision

 Displays the revision of the currently selected repository. Click Search to specify another revision for the repository.

Click Select to have the IDE compare the specified folders and display the differences.
