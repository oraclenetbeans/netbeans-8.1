Add Java Platform wizard: Sources and Javadoc Page

Use to specify the classpaths of source files and Javadoc used by a custom (non-UEI compliant) emulator platform.

Element Description

Sources

 Specifies the classpath of the platform source files.

Click Add to add additional source files. Select a source file and click Remove to remove it from the platform.

Javadoc

 Specifies the classpaths of the Javadoc used by the platform.

Click Add to add associate another Javadoc. Select a Javadoc and click Remove to disassociate it from the platform.
