Options Window: PHP: Frameworks & Tools: Nette Tester

In this panel you can set global options for projects that use the Nette Tester framework.

Element Description

Tester File

 Specify the path to the Nette Tester file. Use the Browse and Search buttons to locate Zend run scripts on your local file system.

php.ini

 Specify the path to php.ini file.
