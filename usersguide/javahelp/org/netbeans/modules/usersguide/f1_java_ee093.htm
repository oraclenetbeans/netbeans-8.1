Web Service Attributes Editor: Quality of Service: Operations

In the Quality of Service tab's Operations section, on the service side, you can add Quality of Service security to a specific web service operation.

You can configure Quality of Service security on the following levels:

Element Description

Operation

 Use this section to specify a transaction setting and a security mechanism for a web service operation.
Input Message Use this section specify parts of a message that require integrity protection (digital signature) and/or confidentiality (encryption).
Output Message Use these sections to specify which message parts have sensitive information that needs to be encrypted and/or signed.
