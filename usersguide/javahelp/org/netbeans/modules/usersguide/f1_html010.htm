Project Properties Window: JavaScript Frameworks: RequireJS

You use this panel to enable support for the RequireJS framework in the editor and specify details for managing dependencies in your Javascript project.

Value Description

Enabled

 Select to enable support for the RequireJS framework in your project.

Path Mapping

    Module

 Type in the text field to specify the name of the module.

    Local Path

 Click Browse to locate the directory or file on your local system.

You can use the following buttons to the right of the table:

Element Description

New

 Click to add a new line to the table.
Remove Select a line with a module name in the table and click Remove to delete it.
