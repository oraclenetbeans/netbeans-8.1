Add Java Platform Wizard: Bootstrap Libraries Page

Use to specify the classpaths of libraries used by a custom (non-UEI compliant) emulator platform.

Element Description

Bootstrap Libraries

 Displays the libraries and APIs used by the platform that the IDE detects.

Click Add and navigate to the library file to add another library. Select a library and click Remove to remove it from the platform.

Detected APIs

 Displays the device profiles, device profiles, and any optional APIs supported by the platform.
