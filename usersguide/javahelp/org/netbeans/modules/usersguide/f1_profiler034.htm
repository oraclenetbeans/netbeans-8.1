Snapshot Window

Use to manage your profiler snapshots.

Element Description

Project

 Select project from the drop down menu

Snapshot

 Displays snapshots

Open Selected Snapshot

 Use to open the snapshot selected in the Snapshot table.

Export Selected Snapshot

 Use to export the snapshot selected in the Snapshot table. A dialog appears to browse your system for the target export location.

Compare Selected Snapshots

 Use to compare to snapshots. This icon is active only when two snapshots are selected in the Snapshot table.

Rename Selected Snapshot

 Use to rename the snapshot selected in the Snapshot table.

Delete Selected Snapshot

 Use to delete the snapshot selected in the Snapshot table.
