Project Properties Window: JavaScript Libraries: CDNJS

The CDNJS page in the Project Properties window enables you to add, edit, update or remove CDNJS libraries from your project.

Value Description

Library

 Displays the name of an installed CDNJS library.

Version

 Displays the version of an installed CDNJS library.

Latest Version

 Displays the most recent version of the library that is available.

Libraries Folder

 Displays the path to the folder where the downloaded CDNJS libraries are placed and installed.

You can use the following buttons to the right of the table:

Element Description

Add

 Click to add a dependency on the specified package.
Edit Click to edit the installed and the required version of the library.

Remove

 Select a CDNJS library in the table and click Remove to remove it.

Update

 Select a CDNJS library in the table and click Update to update the details.
Browse Click to select a directory for CDNJS libraries.
