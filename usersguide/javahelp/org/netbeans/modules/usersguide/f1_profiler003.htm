OQL Console: Query Properties Dialog Box

Use to modify the properties of custom saved OQL queries and to modify the order that custom saved queries are listed under the Custom node in the Saved Queries pane.

Note: You cannot modify the properties or change the order of the default saved queries that are bundled with the IDE and listed under the Samples node or the PermGen Analysis node.

Element Description

Name

 Modify the name of the query.

Description

 (Optional) Modify the description of the query.
