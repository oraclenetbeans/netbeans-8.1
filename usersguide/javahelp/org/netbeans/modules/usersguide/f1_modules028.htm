Update Center Wizard: Update Center Declaration

The New Update Center wizard lets you create a module that registers an update center in the user's Plugins manager.

In effect, the module bundles the URL to the update center so that the user does not need to register the URL manually. You open the Update Center wizard by right-clicking a module project, choosing New > Other and selecting Update Center from the Module Development category.

Element Description

URL to Update Descriptor

 Specifies the URL to the autoupdate descriptor. You can let the IDE generate the autoupdate descriptor for you. How?

Display Name

 Specifies the name that you would like the user to see in their Plugins manager.
