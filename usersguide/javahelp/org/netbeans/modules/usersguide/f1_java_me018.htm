New Embedded Application Wizard: Default Platform Selection Page

Use to choose the initial emulator platform for the project. The emulator platform determines the execution environment for the project.

Note: After you create the project, you can create additional project configurations with alternate platform settings.

Element Description

Emulator Platform

 Specifies the execution environment for the project.

The drop-down menu provides a list of available emulator platforms.

Device

 Specifies a mobile device emulator supported by the selected emulator platform.

When executed, the application's performance is simulated on this device.

Device Configuration

 Specifies the supported device configuration of the emulator platform.

Choices that are not supported by the selected emulator platform are grayed out.

Device Profile

 Specifies the version of MIDP supported by the emulator platform.

Choices that are not supported by the selected emulator platform are grayed out.
