Options Window: Appearance: Document Tabs

You use this panel to customize the NetBeans window system's functionality.

You can set the following options:

Element Description

Document tabs

 Modify the position of tabs in the document area by using the following options:

New document opens next to active document tab

 If selected, a new document opens in the same docking spot next to the active document tab.

Close activates most recent document

 If selected, the most recent document becomes active after an active document is closed.

Tabs placement

 Select where to place the editor tabs. You can choose top, bottom, left or right.

Multi-row tabs

 If selected, the editor tabs display in multiple rows.

Show parent folder name in tab title

 If selected, the parent folder is also displayed in tab (for example, web/index.html).

Same background color for files from the same project

 If selected, the tabs of files in the same project will have the same background color.

Show full file path

 If selected, the full path to the file that is visible in the editor is displayed below the tabs of the open files.

Sort opened documents list by project

 If selected, the list of files open in the editor is sorted by project.
