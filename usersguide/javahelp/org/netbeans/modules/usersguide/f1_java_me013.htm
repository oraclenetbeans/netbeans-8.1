Add Java Platform Wizard: Detected Platforms Page

Use to view all emulator platforms detected on your system. All platforms that are compliant with the Unified Emulator Interface (UEI) standard are marked with a check.

Element Description

Select Platforms to Install

 Displays the install location of the selected platform.

Description

 Displays the devices and device profiles supported by the platform.

Javadocs

 Displays any Javadocs associated with the platform.

Click Add to associate Javadocs or select a listed Javadoc. Click Remove to remove its association with the platform.

Sources

 Displays the source file location of the platform.
