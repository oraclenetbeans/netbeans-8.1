JavaScript Breakpoint Properties: Line

You use this dialog box to set the properties of a JavaScript line breakpoint. You set line breakpoint properties when you set a new line breakpoint or when you modify an existing line breakpoint.

You can open this dialog box by right-clicking a line breakpoint in the Breakpoints window and choosing Properties. You can set the following values in this dialog box.

Element Description

File

 Specifies the name of the source file that contains the breakpoint.

Line number

 Shows the line number on which the breakpoint is set.
