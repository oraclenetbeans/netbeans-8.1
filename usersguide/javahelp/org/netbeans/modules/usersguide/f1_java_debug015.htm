Debugging: Events Window

Use to view a list of logged events and add listeners to help you with debugging the project.

To add a listener for logging purposes, double-click the Custom Listeners node or Event Log node to open the Select Listener dialog box. Select listeners from the list according to the type of event that you want to log.
