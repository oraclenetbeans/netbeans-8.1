New PHP Application from Remote Server: Remote Connection

Specify the following details to retrieve source files from a remote server.

Element Description

Project URL

 Specify the URL of the remote project.

Remote Connection

 Select a connection from the drop-down list. Click Manage to create a new connection or modify the details of an existing connection.

Upload Directory

 Specify the root directory that contains the project sources on the remote server.
