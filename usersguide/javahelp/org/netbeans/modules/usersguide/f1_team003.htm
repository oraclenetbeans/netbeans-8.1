New Team Project Wizard: Summary

The New Team Project wizard enables you to create a project on a registered Team Server. You open the wizard by choosing Team Server > New Team Project from the Team menu. You can share the sources of an open project by right-clicking the project in the Projects window and choosing Share on Team Server in the popup menu.

The Summary page of the wizard enables you to view a summary of the services that will be set up when you click Finish. Depending on the options you selected in the wizard, the Summary page displays the following details about the Team project.

  • the name and location of the project that will be created on Team Server
  • the location of the local repository
  • the location of the remote repository
  • the projects and/or files that will be committed to the remote repository
  • if an issue tracker for the project will be set up
  • if a chat room for the project will be created

Note: If any of the details are incorrect, use the Back button to change the wizard options.

After you click Finish, the IDE will create the project on the Team Server and add the project to the list of My Project in the Team Dashboard. If you selected Commit sources in the wizard, the project sources are committed to the remote repository.

After the project is created, the sources can be downloaded freely by anyone. However, only approved members of the project team can commit changes to the repository and participate in the project's chat room.
