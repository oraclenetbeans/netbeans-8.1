Options Window: PHP: Code Analysis

Use this panel to specify the following global options for PHP code analysis.

Element Description

Code Sniffer

 Select this category to specify the following Code Sniffer options.

    Code Sniffer

 Specify the path to the Code Sniffer script. Click Browse or Search to locate the scripts on your local file system.

    Default Standard

 Select a default PHP standard from the drop-down list.

Coding Standards Fixer

 Select this category to specify the following Coding Standards Fixer options.

    Coding Standards Fixer

 Specify the path to the Coding Standards Fixer script. Click Browse or Search to locate the scripts on your local file system.

    Default Level

 Select a level from the drop-down list to set the limit for the fixers that are applied. This option corresponds to the --level option.

    Default Config

 Select the default configuration from the drop-down list.

    Default Options

 Type specific CS Fixer options in the text field, separated by a comma.

Mess Detector

 Select this category to specify the following Mess Detector options.

    Mess Detector

 Specify the path to the Mess Detector script. Click Browse or Search to locate the scripts on your local file system.

    Default Rule Sets

 Select the rule sets that are run by default.
