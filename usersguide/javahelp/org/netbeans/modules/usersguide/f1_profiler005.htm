New Profiling Point Wizard

Use to set a new profiling point that invokes a profiling action. In this wizard, you specify the type of profiling point that you want to set according to the profiling action that you want to be invoked. Depending on the type of profiling point that you select, specify either a line in your code or a condition that needs to be met to invoke the action. In this wizard, you also specify settings for the action that is performed.

Element Description

Profiling Point Type

 . Select the type of profiling point according to the action that you want to invoke:
  • Reset Result
  • Stopwatch
  • Take Snapshot
  • Load Generator
  • Timed Take Snapshot
  • Triggered Take Snapshot

A description of the selected type is displayed in the wizard.

Profiling Point Project

 Select a project from the dropdown list of open project.
