GUI Builder Connection Wizard: Select Source Event

Use to select the source event.

Specify the following properties in the Connection wizard Select Source Event page:

Element Description

Source Component

 Displays the source component of the triggering event , which is automatically set for you when it is selected in the GUI Builder or Navigator window.

Events

 Selects the event of a source component on which the trigger component will react by expanding the event type directory nodes.

Events Handler Method

 Specifies the default handler method name.

Enter a new name to specify a method other than the default.

Click Next to specify an operation to be performed on the target component.
