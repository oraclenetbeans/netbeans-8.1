Enter Password

Use to enter a password under the following conditions:

  • You are deploying an application and did not specify a password in the Deploying Properties page.

  • You are unlocking a locked keystore file or locked key pair. Some keystore files and individual key pairs within a keystore file are locked with passwords as an additional security method. You have to unlock these keystores and key pairs before you can use them to sign your MIDlet suite.

Element Description

Password

 Specifies the password that unlocks the keystore file.
