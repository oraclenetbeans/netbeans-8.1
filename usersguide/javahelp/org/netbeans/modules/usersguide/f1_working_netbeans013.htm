Options Window: Keymap

You use this panel in the Options window to set the keyboard shortcuts for the IDE. You can change between pre-configured shortcut sets, create new shortcut sets, or change the keyboard shortcut for an individual command.

This panel contains the following elements.

Element Description

Profile

 Select a profile from the list to switch between profiles.

Show as HTML

 Click to view a table with all keyboard shortcuts in your browser.

Manage Profiles

 Click to open the Manage Keymap Profiles dialog box to create, remove, import and export keymap profiles.

Search

 Type in the text field to filter the shortcuts by the name of the command.

Search in Shortcuts

 Type in the text field to filter the list to the shortcuts containing the keyboard character or characters.
