Project Properties Window: Ignored Folders

The Ignored Files panel contains the following elements:

Element Description

Ignored Folders

 This field lists all folders in the project that you want the IDE to ignore for FTP upload, scanning, etc.

Click Add Folder to locate any folders in your project that you want the IDE to ignore.

Extra Folders for Code Analysis

 This field displays a list of folders that you want the IDE to include for code analysis.

Click Add Folder to locate any folders in your project that you want to add.
