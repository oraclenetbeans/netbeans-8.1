Manage Calibration Data Dialog Box

Use to calibrate the IDE with a specific Java platform.

Element Description

Java platforms available for profiling

 This table displays a list of the available local and remote Java platforms and the date of the most recent calibration.

    Java Platform

 This column displays the name of the platform.

You can add or remove Java platforms in the Java Platform Manager.

    Last Calibrated

 This column displays the date of the most recent calibration data for the platform.

No date is displayed if the file containing the calibration data for the platform is missing or corrupted.

Calibrate

 Click to run the calibration process for the selected Java platform.

The button is disabled if you select a remote Java platform.

Java Platforms

 Click to open the Java Platform Manager to add or remove a Java platform.
