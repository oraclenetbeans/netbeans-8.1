New Free-Form Project Wizard: Build and Run Actions

Use to map IDE commands to targets in your free-form project's Ant script.

The drop-down lists for each command (Build Project, Clean Project, Generate Javadoc, Run Project, and Test Project) contain all of the targets in your project's Ant script.

Note: If your Ant script does not contain a target for one of the commands on this page, the command is unavailable in the IDE.
