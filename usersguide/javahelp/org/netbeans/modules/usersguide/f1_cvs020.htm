CVS: View Revision Dialog Box

The CVS View Revision dialog box enables you to choose a specific revision for a versioned file and view that revision in the IDE's main window.

In the View Revision dialog box, choose from the following two options:

Element Description

Tag/Branch/ Revision

 Select this option to specify the name of the tag, branch, or revision you want to view. In the adjacent text box, either type in the keyword that identifies the tag, branch, or revision, or click Browse to view items maintained in the repository.

Working Branch HEAD

 Select this option to view the most current revision of the selected file.
