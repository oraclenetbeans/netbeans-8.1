Project Properties Window: PHPDoc

In this pane you can set the project properties for documentation generated by PHP Documentor. You must have PHP Documentor installed. You can configure the IDE to use PHP Documentor in the Frameworks & Tools tab of the PHP category in the Options window.

Element Description

Target Directory

 The directory containing the generated PHP Documentor documentation.

Click Browse to locate the target directory on your local file system.

Title

 The title of the generated documentation.
