Safe Delete Dialog Box

Use to specify options for the Safely Delete refactoring command.

Element Description

Safely delete

 Lists the code elements that you have selected for deletion.

If the code elements do not correspond with the elements that you intend to delete, click Cancel.

Search in Comments

 Warns you if the code element to be deleted appears in a comment.
