Editor Options: Highlighting

This tab enables you to specify which occurrences of code symbols and keywords will be marked when the cursor is on one of their instances.

This panel contains the following elements.

Element Description

Language

 Select Java or PHP or C/C++.

Mark Occurrences of Symbol Under Caret

 Disable this to universally turn off the marking of keywords.

Specific Keywords (Java and C/C++)

 With Mark Occurrences enabled, select the categories of keywords whose occurrences you want to mark. Select Keep Marks for the marks to persist after you move the caret.
