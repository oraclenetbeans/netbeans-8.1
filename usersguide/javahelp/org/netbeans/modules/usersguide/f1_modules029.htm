Window Wizard: Name, Icon and Location

The New Window wizard creates a new window component. More

You open the New Window wizard by right-clicking a module project, choosing New > Other and selecting Window from the Module Development category.

Element Description

Class Name Prefix

 Specifies the prefix of the new Action and TopComponent classes.

Icon

 Specifies the icon that will accompany the window component. For example, you will see the icon in the label of the window component. (Optional)

Package

 Specifies the name of the package where the class will be housed.
