Connect to Database Dialog

Use to supply the user name and password to open an existing database connection. You open this dialog by right-clicking the database connection node in the Services window and choosing Connect.

Element Description

User Name

 Enter the user name for the database.

Password

 Enter the password for the database.

Remember Password

 Select if you want the IDE to remember your password.

If selected, you will not be prompted for the user name and password on subsequent connections to the database and the Remember Password property is set to True.
