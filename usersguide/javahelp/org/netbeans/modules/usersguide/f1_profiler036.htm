Profile Window with Telemetry

Use to customize and capture data during a telemetry session.

The following table shows the available actions.

Element Description

CPU and GC

 Toggle this icon to show and hide the CPU and GC graphic.

Memory

 Toggle this icon to show and hide the Memory graphic.

Garbage Collection

 Toggle this icon to show and hide the Garbage Collection graphic.

Threads and Classes

 Toggle this icon to show and hide the Threads and Classes graphic.

Thread Dump

 Creates a textual dump of all active threads and monitors of the profiled application.

Heap Dump

 Saves an image of the current heap content of the profiled process in .hprof format.

GC

 Requests the JVM of the profiled process to invoke garbage collection.
