Configuring Code Assistance Settings for C and C++ Projects

See Also 

You can use the Options window to configure default settings for your C and C++ projects.

To configure C and C++ code assistance settings:

  1. Choose Tools > Options.
  2. Click C/C++ in the top pane of the window.
  3. Click the Code Assistance tab.
  4. On the Code Assistance tab, select the appropriate compiler tab. Modify the settings for include files and macro definitions that are used by the built-in parser.
