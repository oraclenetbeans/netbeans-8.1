Setting Debugging Options

See Also 

You can set some debugging options specifically for a project. For information on setting global debugging options that will persist for all projects, see Setting Global Debugging Options.

To set debugging options for a project:

  1. Right-click the project node and choose Properties, or select the project node and choose File > Project Properties.
  2. In the Project Properties dialog box, select the project configuration for which you want to set debug options.
  3. Select the Debug node and modify the options.

See C/C++ Project Properties Dialog Box: Debug for information about the options.
