What You Should Know Before Working With the Java Card 3 Plugin

Developers can use this IDE and the Application Programming Interface, Java Card Platform, Version 3.0.1, Connected Edition to implement web applications and applets with extended and enhanced capabilities for the Java Card platform. Before beginning development of these applications and applets, developers should become familiar with the Java programming language, object-oriented design, the Java Card technology specifications, and smart card technology. A good resource for becoming familiar with Java and Java Card technology is the web site located at:

http://java.sun.com/javacard
