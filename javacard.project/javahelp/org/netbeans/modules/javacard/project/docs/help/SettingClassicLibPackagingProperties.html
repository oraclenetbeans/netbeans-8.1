Setting Classic Library Build - Packaging Properties

The Build - Packaging panel enables you to edit the Classic Package AID values. The Classic package AID is a unique ID applied to the package of the classic applet. It consists of two components, an RID value and a PIX value.

The RID value is the unique ID for your organization, assigned by a standards body such as the International Standards Organization (ISO). The RID is a 5 byte hexadecimal value. A default RID is provided by the plugin and can be used during development. However, before deployment you must obtain a valid RID from the ISO and add it to the project.

The PIX (proprietary identifier) value uniquely identifies an individual Java Card package. It is a 0 - 11 byte hexadecimal value generated by the IDE. A specific value is not required for development. To generate a PIX value for the package, click the Generate button.
