Getting Updated Samples

You can use the IDE to get the latest working samples from the Kenai Java Card 3 repository. Documentation for configuring and running these samples is also located in the Kenai repository.

  1. In the IDE menu, choose Team > Kenai > Get Sources from Kenai.
  2. In the Get Sources from Kenai dialog box, click the Browse button beside the Kenai Repository field and use search to locate the "Java Card - Everything You Need to Know" project.

    Note - There are two Java Card 3 platform repositories, one that uses Subversion version control and another that uses Mercurial version control.

  3. Select the project that you want and then click the OK button.
  4. In the Get Sources from Kenai dialog box, click the Browse button beside the Folder to Get field, select the "samples" folder in the Browse dialog box and then click the OK button.
  5. Accept the default Local Folder location or set your own preference.
  6. Click the Get From Kenai button.
  7. The IDE downloads the sample projects from the Kenai site and opens a dialog box that lets you select the one or ones that you want to open from the list of downloaded projects.
