Add an Instance of an Applet

  1. Click Customize Instances.

    The IDE opens the Instances dialog box.

  2. Click Add and the IDE adds the new instance to the list under Applet Instance.
  3. Add special deployment parameters in the Deployment Parameters field.
  4. Accept the Instance AID value generated by the IDE or generate a new Instance AID.

    When the Instances dialog box closes, the Applets panel displays the new applet instance in the Instances To Create on Run list.
