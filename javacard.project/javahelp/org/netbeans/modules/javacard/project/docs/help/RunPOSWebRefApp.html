Run POSWeb

  1. Click File > Open Project in the main window.
  2. Go to the JC_CONNECTED_HOME\samples\reference_apps\Transit folder and open the POSWeb project.
  3. Select the POSWeb project and click the run button in the main window.

    The browser opens and displays the transit application entry page.

  4. Click the Continue button and in the login screen, enter the User Name and PIN.

    User Name: owner-pos

    PIN: 8888

    Note - If the browser displays a warning that the security certificate is not issued by a trusted certificate authority, disregard it and choose to continue to the web site. The security certificate for this sample is used for demonstration purposes only and cannot be used for developing deployable samples.If you are unable to continue to the web page, perform the procedure described in Accepting Untrusted Certificates.

    The browser displays the transaction page.

  5. Click the CREDIT button.

    The browser displays the Credit Request page.

  6. Enter a number in the field and click the SUBMIT button.

    The page displays the credited amount.

  7. Click the AUTHORIZE button.
  8. In the login screen, enter the User Name and PIN.

    User Name: owner-admin

    PIN: 8888

    This authorizes access to the Admin application from a non-trusted client (the Internet Explorer browser in this RI). Clicking the CANCEL button (when enabled) cancels remote authorization.
