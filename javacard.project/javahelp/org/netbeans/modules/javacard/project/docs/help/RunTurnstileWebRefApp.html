Run TurnstileWeb

  1. Click File > Open Project in the main window.
  2. Go to the JC_CONNECTED_HOME\samples\reference_apps\Transit folder and open the TurnstileWeb project.
  3. Select the TurnstileWeb project and click the run button in the main window.

    The browser opens and displays the Transit Turnstile Home Page.

  4. Click the Continue button.

    The browser displays the Entry page.
