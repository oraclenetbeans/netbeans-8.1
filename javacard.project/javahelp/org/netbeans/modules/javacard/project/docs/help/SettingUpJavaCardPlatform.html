Setting Up a Java Card Platform

When you install the Java Card Runtime Bundle plugin, it automatically sets up a default Java Card 3 platform and associated default device in the IDE. Before working in the IDE to create a Java Card platform application you might want to verify that the platform is set up for your use. To verify that the platform is set up correctly, see Verify the Java Card Platform is Set Up.

If you use a different or a custom Java Card platform that you want to create applications for, you must add it to the IDE. See Add a Java Card Platform. If you create applications to run on a device with different settings than those of the default device, you must add a new device or customize the existing device with the appropriate settings. See Add a New Device or Customize a Device.
