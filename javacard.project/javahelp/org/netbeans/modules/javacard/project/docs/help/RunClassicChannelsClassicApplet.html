Run ClassicChannels Classic Applet

  1. Click File > Open Project in the main window.
  2. Go to the JC_CONNECTED_HOME\samples\classic_applets folder and open the ClassicChannels project.
  3. Click the run button in the main window.

    The IDE compiles, builds, and runs the sample. When the sample runs, it creates an output file.

  4. Verify that the contents of the output file created by the applet are the same as the contents of the ClassicChannels.expected.out file.
