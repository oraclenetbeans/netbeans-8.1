Unable to Run a Sample

The samples that you are using might not be current. Older samples might be incompatible with the current plugin and Java Card RI. The latest samples for the Java Card 3 platform and the instructions for running them are located in the Kenai repository. You can down load any or all of the latest samples directly from within the IDE.

  • See Getting Updated Samples for instructions on how to download updated samples from the Kenai project site.
  • For the latest information about running and troubleshooting the samples, go to the Java Card 3 Kenai project site, http://kenai.com/projects/javacard.
