Create a Project

Use the IDE New Project wizard to generate a project from the sample sources and templates provided by the Java Card plugins. The types of Java Card projects that can be created are:

  • Classic Applet - see Creating a Classic Applet Project

  • Extended Applet - see Creating an Extended Applet Project

  • Classic Library - see Creating a Classic Library Project

  • Extension Library - see Creating an Extension Library Project

  • Web Project - see Creating a Web Application Project

The wizard creates the required directories, resources, and descriptor files from settings that you provide in the wizard and from the setting of the Java Card platform and device set up in the IDE. See Setting Up a Java Card Platform.
