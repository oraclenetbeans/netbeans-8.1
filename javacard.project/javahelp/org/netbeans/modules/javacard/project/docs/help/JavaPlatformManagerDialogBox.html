Java Platform Manager Dialog Box

The Java Platform Manager dialog box enables you to perform the following:

  • Add or remove Java Card Runtime (JCRE) platforms.

  • Associate a device or card with the platform.

  • Set the class path of the Java Card Connected version API JAR file.

  • Set the path of the API sources.

  • Set the path of the API Javadoc.

  • Set the class path of the Java Card Classic version API JAR file.
