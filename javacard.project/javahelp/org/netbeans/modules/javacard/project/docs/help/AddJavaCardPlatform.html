Add a Java Card Platform

  1. Choose Tools > Java Platforms from the main window.
  2. Click Add Platform in the Java Platform Manager dialog.
  3. Select Java Card Platform and click Next.
  4. Select the directory that contains the new or custom Java Card platform. Java Card platform directories are marked with a (image of icon) in the file chooser.
  5. Click Next to validate the platform.

    Note - You can give the platform a custom name in the Display Name field.

  6. Click Next to define a new Default Device for the new platform or Finish to use the existing default device with the new platform.

    If a new default device is set up for a platform, both it and the Default Device are associated with the platform. After adding the platform, you can use the Device Manager dialog to add or remove devices for a platform.
