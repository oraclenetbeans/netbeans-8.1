Java Card 3 Project Properties Dialog Box

To open the Project Properties dialog box, either right-click the project in the Project window and choose properties or select the project in the Projects window and click File > Project Properties in the main window.

The properties in the Project Properties dialog box are grouped in the following categories and displayed on separate panels:

  • Sources Panel

  • Dependencies Panel

  • Applets Panel (only used for classic and extended applet projects)

  • Build - Compiling Panel

  • Build - Packaging Panel (only used for classic applet and classic library projects)

  • Build - Security Panel

  • Run Panel
