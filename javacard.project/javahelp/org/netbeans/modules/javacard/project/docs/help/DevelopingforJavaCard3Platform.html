Developing for the Java Card 3 Platform

The IDE provides developers with Java Card 3 plugins that add a rich set of features to the NetBeans IDE enabling developers to easily create and run applications for the Java Card 3 platform Connected Edition:

  • Support for developing all application types of Java Card 3 platform Connected Edition, including editing, building, deploying, and running.

  • Support for multiple reference implementation (RI) configurations.

    Individual projects can be associated with different RIs configured with different options such as RAM size and EEPROM size.

  • A built-in copy of the Java Card 3 Reference Implementation used in developing Java Card projects.

    Templates and sample projects of all application types enable developers to rapidly create their own custom applications for the Java Card 3 platform Connected Edition.
