What Is In This Online Help

This online help provides information for developers about using the IDE to create applications for the Java Card 3 platform. The contents of this online help are organized into three major sets of topics:

Developing for the Java Card 3 Platform

Describes the basic sequence of development activities performed by a developer, how to set up a Java Card platform in the IDE for development activity, how to set up security in the IDE for client-server Java Card applications, and how to use the IDE to debug applications.

Troubleshooting

Describes how to troubleshoot common problems experienced when using the IDE to create or run Java Card applications.

Java Card 3 Glossary

Contains Java Card platform terms and their definitions.
