Getting Additional Web Application Samples

You can use the IDE to get additional working web application samples from the Kenai Java Card 3 repository. Documentation for configuring and running the web application samples is also located in the Kenai repository.

  1. In the IDE menu, choose Team > Kenai > Login to Kenai.

    2. Note - If you do not have an existing account on Kenai you can create one from the Login to Kenai dialog box.

  2. Login to the Kenai project site.
  3. In the IDE menu, choose Team > Kenai > Get Sources from Kenai.
  4. In the Get Sources from Kenai dialog box, click the Browse button beside the Kenai Repository field and use search to locate the "Java Card - Everything You Need to Know" project.

    Note - There are two Java Card 3 platform repositories, one that uses Subversion version control and another that uses Mercurial version control.

  5. Select the desired "Java Card - Everything You Need to Know" project and then click the OK button.
  6. In the Get Sources from Kenai dialog box, click the Browse button beside the Folder to Get field, select the samples/web folder in the Browse dialog box and then click the OK button.
  7. Accept the default Local Folder location or set your own preference.
  8. Click the Get From Kenai button.
  9. The IDE downloads the sample web application projects from the Kenai site and opens a dialog box that lets you select the web project or projects that you want to open.
